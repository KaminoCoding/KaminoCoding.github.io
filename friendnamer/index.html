Friend Namer

This is a small mod I wrote up in a couple hours. It allows you to add custom nicknames to your friends. I will release a tutorial video eventually but until then there are text instructions. This is a clientside only mod so it won't effect anything on a server and should be compatible with all and every server wherever they are played.

You can access the options by clicking the Mods button on the main menu the clicking on the name of the mod and clicking the Config button underneath that. The mod is in beta so please give it a try and tell me whats going wrong or what you want to see it have.
